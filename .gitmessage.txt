# Types:
# feat: 새로운 기능 추가
# fix: 버그 수정
# docs: 문서 수정
# style: 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우
# refactor: 코드 리팩토링
# test: 테스트 코드 추가 또는 수정
# chore: 빌드 업무 수정, 패키지 매니저 수정

# Scopes:
# app: 앱 전체
# ui: 사용자 인터페이스
# auth: 인증 관련
# api: API 관련
# db: 데이터베이스 관련
# config: 설정 관련

# Examples:
# feat(auth): 로그인 기능 추가
# fix(ui): 버튼 클릭 이벤트 수정
# docs: README 업데이트
# style: 코드 포맷팅 정리
# refactor(api): API 호출 로직 개선
# test: 로그인 컴포넌트 테스트 추가
# chore: 의존성 패키지 업데이트
